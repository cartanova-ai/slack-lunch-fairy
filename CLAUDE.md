# CLAUDE.md

이 문서는 Claude Code가 이 프로젝트에서 작업할 때 따라야 할 지침입니다.

## 프로젝트 개요

**점심 요정 (Lunch Fairy)** - Slack 점심메뉴 알림 봇

## 보안 유의사항

**이 저장소는 PUBLIC입니다. 커밋/푸시 전에 반드시 확인하세요.**

### 절대 커밋하면 안 되는 것들
- Slack Bot Token, App Token
- API 키, 비밀번호, 인증 정보
- .env 파일 (반드시 .gitignore에 포함)
- 개인정보가 포함된 데이터
- 내부 서버 주소, IP

### 커밋 전 체크리스트
1. `git diff --staged`로 변경사항 검토
2. 하드코딩된 토큰/키가 없는지 확인
3. 환경변수로 처리해야 할 값이 코드에 직접 들어가지 않았는지 확인
4. .env.example은 실제 값 없이 키 이름만 포함

### 환경변수 관리
- 민감한 정보는 모두 환경변수로 관리
- .env 파일은 .gitignore에 포함
- .env.example 파일로 필요한 환경변수 목록 제공

## 코드 작성 원칙

- 한국어 주석 사용
- 명확하고 간결한 코드
- 불필요한 복잡성 지양
